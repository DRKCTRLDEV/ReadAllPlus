# ReadAllPlus

**ReadAllPlus** is a Discord/Vencord theme that enhances the "ReadAll" Vencord plugin by replacing the default 'Read All' text label with a sleek, centered checkmark (/Clipboard/DoubleCheckmark/Envelope/Bell) icon. The button is styled to match Discord's tutorial container buttons, such as the Direct Messages button, and features a lighter background for better visibility. This theme adds smooth animations and a lovely accent, making it a seamless and visually appealing addition to the Discord UI.

## Features

- Replaces the 'Read All' text with a centered checkmark icon.
- Styled to match Discord's native UI, including a lighter background and rounded edges.
- Animated button with a smooth hover effect and a bounce animation on click.
- Fits seamlessly into Discord's interface, providing a polished and cohesive look.

## Requirements

- [Vencord](https://github.com/Vendicated/Vencord) must be installed.
- The [ReadAll plugin](https://github.com/Vendicated/Vencord) must be enabled.

## Installation

1. Ensure you have [Vencord](https://github.com/Vendicated/Vencord) installed and the [ReadAll plugin](https://vencord.dev/plugins#) enabled.
2. Install the "ReadAll" Notifications plugin from vencords repository,
3. Download the latest version of the ReadAllPlus theme from the [GitHub repository](https://github.com/drkctrldev/readallplus/releases).
4. Add the theme to your Vencord installation by following the instructions in the [Vencord documentation](https://docs.vencord.dev/).
